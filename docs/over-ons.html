1. Wat is Tweakly?

Tweakly is een mobiele prijsvergelijkingsapp voor elektronica, gericht op de Nederlandse markt. De app vergelijkt prijzen bij populaire webshops zoals Coolblue, Bol.com, MediaMarkt, Amazon.nl, Alternate en meer.

Tweakly is geïnspireerd op de functionaliteiten van bekende Nederlandse tech-sites, maar met een moderne mobiele ervaring als uitgangspunt. De focus ligt op:

  • Eerlijke prijsvergelijking (gesorteerd op prijs, niet op commissie)
  • Prijsgeschiedenis en -trends per product
  • Prijsalerts via pushnotificaties
  • Technisch nieuws via RSS-feeds van Tweakers.net
  • Productreviews en specificaties

2. Onze waarden

⚖️

Eerlijkheid

Winkels worden altijd gesorteerd op prijs. Commissiehoogte heeft geen invloed op rangschikking.

🔗

Transparantie

Affiliate-links worden duidelijk gemarkeerd. We zijn open over hoe we inkomsten genereren.

💰

Geen meerkosten

U betaalt nooit meer via een affiliate-link. De commissie wordt door de webshop betaald.

🔒

Privacy

We verzamelen minimale data en volgen de AVG-richtlijnen. Geen verkoop van gebruikersdata.

3. Technologie

Tweakly is gebouwd met moderne technologie:

  • App: React Native met Expo SDK 54 (iOS & Android)
  • Backend: Supabase (PostgreSQL, Auth, Storage)
  • Hosting: Expo Application Services (EAS)
  • Website: Statische HTML, gehost op Netlify
  • Productdata: Icecat Open Catalog voor specs en afbeeldingen
  • Nieuwsfeeds: Tweakers.net RSS via FeedBurner

4. Affiliate-relaties

Tweakly bevat affiliate-links naar externe webshops. Wanneer u via Tweakly een aankoop doet, kan Tweakly een kleine commissie ontvangen van de betreffende webshop. Dit heeft geen invloed op de prijs die u betaalt.

Lees onze volledige Affiliate Disclosure voor meer informatie over onze partners en principes.
